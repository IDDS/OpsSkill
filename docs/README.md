# OpsSkill Research Docs Index

Start here if the goal is a paper rather than only a prototype.

## Reading order

1. `research_agenda.md`
   - defines the problem, research gaps, and core innovations
2. `experiment_plan.md`
   - defines tasks, baselines, metrics, and ablations
3. `paper_outline.md`
   - defines the full paper structure
4. `related_work_notes.md`
   - helps write the related work section precisely
5. `acm_ieee_related_work.md`
   - ACM/IEEE-focused literature notes and citation suggestions
6. `related_work_zh_draft.md`
   - a ready-to-edit Chinese related work draft
7. `citation_checklist.md`
   - must-cite and section-by-section citation checklist
8. `end_to_end_intelligent_ops_workflow_zh.md`
   - end-to-end intelligent ops workflow taxonomy, skill seeds, papers, and tools
9. `introduction_zh_draft.md`
   - Chinese introduction draft
10. `problem_formulation_zh_draft.md`
   - Chinese problem formulation draft
11. `method_zh_draft.md`
   - Chinese method draft
12. `abstract_zh_draft.md`
   - provides a ready-to-refine Chinese abstract
13. `contribution_zh.md`
   - lists the main contributions in concise form
14. `task_benchmark_design.md`
   - guides benchmark construction and hidden validation
15. `paper_writing_plan.md`
   - gives a practical weekly writing and experiment schedule
15. `ops_workflow_skill_catalog_zh.md`
   - end-to-end detection, diagnosis, and recovery skill catalog
16. `skill_stage_mapping_zh.md`
   - stage-by-stage mapping from workflow to skill types
17. `experiment_ready_skills.md`
   - experiment metadata format for paper-oriented skill cards
18. `method_innovation_summary_zh.md`
   - concise summary of method novelty and theory-style propositions
19. `agent_planner_zh.md`
   - manager agent and heuristic/LLM planner design notes

## Recommended first-paper scope

For a strong first submission, prioritize three contributions:
- typed Ops skill representation
- hidden verification on remote Kubernetes
- policy-gated safe execution

Treat structured optimization and cross-cluster transfer as optional extensions if time allows.
