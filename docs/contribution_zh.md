# 论文贡献点（中文）

建议主打以下三个核心贡献：

1. **提出面向运维任务的结构化 Skill 表示**
   将运维任务从自由命令生成提升为包含前置条件、动作、成功判据、回滚与风险标签的 Typed Skill IR，为自治运维提供可验证、可审计、可复用的统一抽象。

2. **提出面向远程 Kubernetes 的隐藏验证与安全执行框架**
   在真实远程集群上引入多信号隐藏验证机制，并通过策略门控、能力探测和风险预算控制技能执行过程，使自治运维系统在真实环境中具备更高的可靠性与安全性。

3. **提出基于结构化执行报告的技能优化机制**
   从执行失败中提取结构化失败签名，驱动技能迭代优化，相比纯文本反思更适合复杂、状态化且高风险的运维环境。

可选扩展贡献：

4. **构建 OpsSkill-Bench 或小规模任务集**
   形成面向 Kubernetes 运维任务的标准化评测集合，为后续工作提供统一实验基准。
