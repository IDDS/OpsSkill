# Paper Writing Plan

## Week 1
- finalize paper positioning
- freeze 3 core contributions
- finish related work notes
- build first 10 benchmark tasks

## Week 2
- implement hidden verifier
- implement safety gate and report collection
- run pilot experiments on 5-10 tasks

## Week 3
- run baseline experiments
- run ablations
- collect failure cases and qualitative examples

## Week 4
- write introduction, method, and setup sections
- produce main result tables and figures
- draft abstract and conclusion

## Week 5
- revise related work and discussion
- tighten contribution claims against baselines
- polish figures, captions, and appendix

## Practical Advice
- do not wait for all experiments to finish before writing
- keep a failure case log from day one
- preserve every execution report for later analysis and appendix material
